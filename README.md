# aichatbotfree Theme

This repository contains only the `aichatbotfree` WordPress theme files. Clone or copy this folder into `wp-content/themes/aichatbotfree/` on your WordPress installation to use it.

## Contents
- Theme templates (front page, archives, single posts, chatbot tool CPT)
- Theme assets (CSS)
- ACF JSON exports for the AI Chatbot Article Builder

## Usage
1. Place this folder inside your WordPress `wp-content/themes/` directory.
2. Activate the **AI Chatbot Free** theme from the WordPress admin.
3. Ensure Advanced Custom Fields Pro is installed to use the flexible content layouts defined in `acf-json/`.
